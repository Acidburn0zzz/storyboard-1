To see client and server logs:
  1. Make sure you've installed the Storyboard DevTools extension
  2. Open the Developer Tools from the Chrome menu at the top-right of your browser (More Tools > Developer Tools)
  3. Navigate to the Storyboard tab
  4. Log in (for this example, every login/pwd is fine)
